  Hospital Data Analytics Dashboard

   Overview

This project analyzes simulated hospital operational and financial data to generate insights that support healthcare management and IT audit processes.

The dashboard provides visibility into patient activity, insurance approvals, and revenue distribution across different payment methods.

This project was inspired by practical exposure to healthcare systems during my internship and demonstrates how data analytics supports decision-making and audit transparency.

---

   Objectives

   Analyze patient registration trends over time
   Monitor insurance claim approvals
   Evaluate revenue distribution across insurers
   Understand payment methods (cash, card, insurance)

---

   Dataset Description

The dataset contains simulated hospital transaction data with the following fields:

   Date
   Patient ID
   Department
   Insurance Provider
   Payment Method
   Amount
   Approval Status

---

   Tools Used

   Microsoft Excel (data preparation)
   Power BI (dashboard visualization)

---

   Expected Insights

   Daily patient registration patterns
   Revenue contribution by insurance providers
   Approval trends for insurance claims
   Payment method distribution
